Для компиляции необходимо вызвать из корня:
```
go go get -u -t -v ./...
go build -v -o bin/go-web-search
```

Для прогона всех тестов запустить в корне проекта:
```
go test ./...

```

Для разработки запустить в корне проекта:
```
goconvey

```
